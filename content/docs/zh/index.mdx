# 文档

欢迎使用 **AI Favicon Extractor** - 最全面的 favicon 提取工具。

## 快速开始

几分钟内开始使用 AI Favicon Extractor:

1. 输入网站 URL
2. 点击"提取 Favicons"
3. 下载所有 favicons 为 ZIP 文件

## 功能特性

- 🔍 **深度提取**: 从任何网站提取所有 favicons
- 📊 **视觉矩阵**: 在精美的网格中查看所有提取的图标
- 📦 **一键下载**: 将所有图标下载为 ZIP 文件
- 🤖 **智能分析**: 自动分析 favicon 质量和格式

## API 集成

了解如何将 AI Favicon Extractor 集成到您的应用程序中。

```typescript
import { fetchFavicons } from '@/lib/api';

const icons = await fetchFavicons('https://example.com');
```
